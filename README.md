# Sales Performance & Business Insights

## Tools Used
- Python (Pandas)
- PostgreSQL (SQL)
- Power BI

## Project Description
End-to-end sales analytics project where raw retail data is cleaned using Python,
stored and analyzed in PostgreSQL, and visualized using Power BI to derive business insights.

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin %
- Total Orders
- Monthly Sales Trend
- Profit by Region & Category
- Top & Bottom Products by Profit

## Business Insights
- West region is the most profitable.
- Technology category has the highest margin.
- High discounts negatively impact profit.
- Several products consistently generate losses.

## Files
- python/data_cleaning.py – Data cleaning script
- sql/analysis_queries.sql – SQL analysis queries
- powerbi/Sales_Dashboard.pbix – Power BI dashboard
